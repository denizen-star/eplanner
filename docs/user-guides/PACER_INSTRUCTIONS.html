Guide for Pacers: Creating a Run

Welcome to the Gay Run Club platform! This guide will walk you through the process of creating and managing a run event. As a pacer, you'll coordinate runs for our community, and this platform makes it easy to organize, share, and manage your events.

Overview

When you create a run, you'll receive two important links:

  • Signup Link: Share this with runners so they can register for your run
  • Management Link: Keep this private - use it to view who has signed up

Step-by-Step Instructions

Step 1: Navigate to the Coordinate an Event Page

  1. Open your web browser and go to the Gay Run Club website
  2. On desktop: Click on "Coordinate an Event" in the navigation menu at the top of the page
  3. On mobile: Tap the hamburger menu (‚ò∞) icon, then tap "Coordinate an Event"
Note The platform is fully mobile-friendly. On mobile devices, you'll see a hamburger menu icon (‚ò∞) in the top right corner. Tap it to access the navigation menu.

[SCREENSHOT: Mobile view showing hamburger menu icon]

Step 2: Fill Out the Run Details Form

You'll see a form with several fields to complete. Here's what each field means:

Pacer Name *

  • Enter your full name as you'd like it to appear to runners
  • Example: "John Smith" or "Alex Martinez"
  • This field is required
[SCREENSHOT: Form showing the Pacer Name field]

Run Title (Optional)

  • Give your run a descriptive name if you'd like
  • Examples: "Morning Beach Run", "Sunset Run", "Weekly Group Run"
  • This field is optional - you can leave it blank
  • If provided, the title will appear in the signup page and help runners identify your run
[SCREENSHOT: Run Title field showing example title]

Date and Time *

  • Click on the date/time field to open a calendar picker
  • Select the date and time for your run
  • The default is set to tomorrow at 6:30 PM, but you can change it
  • This field is required
[SCREENSHOT: Date/time picker showing how to select a date and time]

Max Number of Participants *

  • Enter the maximum number of runners you want to allow for this run
  • The default is 10 participants
  • You can increase or decrease this number based on your preference
  • This field is required
[SCREENSHOT: Max Participants field with default value of 10]

Location *

  • Enter the address or location name where the run will start
  • You can enter:
    • A full address (e.g., "1664 Lincoln Rd, Miami Beach, FL 33139")
    • A location name (e.g., "Gaythering")
    • Any recognizable location description
  • As you type, the system will validate the location and show a map
  • This field is required
[SCREENSHOT: Location field with example address entered]

Step 3: Verify Location on Map

After entering the location, a map will appear below the location field showing:

  • A pin marking the exact location
  • The validated address (which may differ slightly from what you typed)
  • A confirmation box showing the validated address
Important Review the map and validated address to ensure it's correct. If the location is wrong, you can edit the location field and try again.
[SCREENSHOT: Map display showing the location pin and validated address confirmation]

Step 4: Submit the Form and Receive Your Links

  1. Once all fields are filled out correctly, click the "Create Run" button
  2. The form will process your submission (this may take a few seconds)
  3. After successful submission, you'll see a success message with two important links
[SCREENSHOT: Success message showing "Run Created Successfully!" with the two links displayed]

Your Links:

Signup Link:
  • This is the link you'll share with runners
  • Copy this link and share it via WhatsApp, email, social media, or any other method
  • Runners will use this link to register for your run
Management Link:
  • This is your private link to manage the run
  • Keep this link secure and don't share it publicly
  • Use this link to view who has signed up, check the number of spots remaining, and see run details
[SCREENSHOT: Close-up of the two links with Copy buttons visible]

Step 5: Share the Signup Link with Runners

Now that you have your signup link, share it with potential participants:

  1. Click the "Copy" button next to the Signup Link
  2. The link will be copied to your clipboard
  3. Share it through:
    • WhatsApp groups
    • Instagram stories or posts
    • Email
    • Facebook events
    • Any other communication channel your community uses
Tip You can also click directly on the link to open it in a new tab and verify it works correctly.
[SCREENSHOT: Example of sharing the link via WhatsApp or social media]

Step 6: Access Quick Guides

On the Coordinate an Event page, you'll find a collapsible "User Guides" section at the bottom. This provides a quick reference guide for creating events. Click on it to expand and see the step-by-step summary.

[SCREENSHOT: User Guides collapsible section on the coordinate page]

Step 7: Use the Management Link to View Signups

To check who has signed up for your run:

  1. Click on your Management Link (or copy and paste it into your browser)
  2. You'll see a page showing:
    • Run details (location, date/time, pacer name)
    • Current number of signups vs. maximum participants
    • A map showing the run location
    • A collapsible list of all runners who have signed up
[SCREENSHOT: Management page showing run details and signups list]

Viewing Signups:

  1. Click on the "Signups" section to expand it
  2. You'll see a list of all registered runners with:
    • Their name
    • Phone number (clickable to call)
    • Email address (if provided)
    • Instagram handle (if provided)
    • Signup date and time
    • Waiver acceptance status
  3. Click the "Refresh" button to update the list with the latest signups
[SCREENSHOT: Expanded signups list showing runner details]

  • Give your run a memorable name
  • Help runners identify your run when sharing
  • Make your run stand out in lists and reports
  • Examples: "Morning Beach Run", "Sunset Social Run", "Weekly Group Run"

Tips for Successful Run Coordination

Before Creating Your Run

  • Plan ahead: Create your run at least a few days in advance to give runners time to sign up
  • Choose a good location: Pick a location that's easy to find and safe for running
  • Set realistic limits: Consider your ability to manage the group when setting max participants
  • Check the weather: While you can't control it, be aware of forecasted conditions

After Creating Your Run

  • Share early and often: Post your signup link multiple times as the run date approaches
  • Monitor signups: Check your management link regularly to see how many people have signed up
  • Communicate with runners: Consider creating a WhatsApp group for the run to coordinate details
  • Be prepared: Arrive early to the location and be ready to welcome runners

Best Practices

  • Be inclusive: Welcome runners of all skill levels
  • Safety first: Ensure the route is safe and well-lit if running in the evening
  • Have a backup plan: Know what to do if weather conditions are unsafe
  • Stay organized: Keep track of who's coming using your management link

Troubleshooting

The map isn't showing my location correctly

  • Try entering a more specific address
  • Include city and state in your location description
  • Check the validated address shown - it may be more accurate than what you typed

I can't find my management link

  • Check your browser history - the link should be there after you create the run
  • If you're an administrator, you can also find your run in the Admin Dashboard and access the management link from there

The form won't submit

  • Make sure all required fields (marked with *) are filled out
  • Check that your date/time is in the future
  • Ensure your max participants is a number greater than 0
  • Verify your location is valid (the map should show it)

I need to change run details after creating it

  • If you're an administrator, you can edit runs through the Admin Dashboard
  • Otherwise, you may need to create a new run with the corrected information
