# Press Release: Gay Run Club Platform Updates (Short Version)

**FOR IMMEDIATE RELEASE**

## Gay Run Club Launches Enhanced Platform Features

**Miami Beach, FL** - Gay Run Club today announced exciting platform enhancements designed to improve user experience and community engagement.

### New Features

**Run Titles** - Pacers can now give their runs descriptive names, making events more memorable and shareable.

**Enhanced Mobile Experience** - Fully responsive navigation with intuitive mobile menu, ensuring seamless access on all devices.

**Quick Reference Guides** - Inline, collapsible guides provide instant step-by-step instructions directly on each page.

**Advanced Reporting** - Administrators now have access to comprehensive Run Reports and Runners Reports with sorting capabilities, tracking participation and engagement metrics.

**Social Sharing** - Signup links automatically generate rich preview cards when shared on social media platforms.

**Privacy Transparency** - Enhanced privacy notices clearly communicate data usage policies.

### Community Impact

These updates reflect our commitment to creating the best possible experience for our LGBTQ+ running community. The platform continues to prioritize safety, inclusivity, and community building.

### Availability

All new features are now live and available to all users. The platform remains free to use for the LGBTQ+ running community.

### About Gay Run Club

Gay Run Club is a community event management platform designed specifically for coordinating and managing running club events for the LGBTQ+ community. With built-in safety features and inclusive design, the platform helps create a welcoming space for LGBTQ+ runners of all levels to connect, stay active, and build lasting friendships.
